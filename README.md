#B2c商城简介：

B2c商城是一个综合性的B2C平台，包含各种相关模块，在本项目中只有部分业务，记录在电商项目中遇到的新东西

# 包含技术(持续更新中：侧重后台技术运用)

* Spring、SpringMVC、Mybatis
* JSP、JSTL、jQuery、jQuery plugin、EasyUI、KindEditor（富文本编辑器）、CSS+DIV---->没有
* Redis（缓存服务器）
* Solr（搜索）
* httpclient（调用系统服务）
* Mysql
* Nginx(web服务器)
* netty(websocket)

* Maven 3.3.3（开发工具自带）
* Tomcat 7.0.53（Maven Tomcat Plugin）
* JDK 1.8
* Mysql 5.6
* Nginx 1.8.0
* Redis 3.0.0
* Win10 操作系统
* GIT（版本管理)
